## 10. Core Feature Flows

### 10.1 Course Creation (Educator)

```
1. Educator accesses /educator/courses/create
2. Fills course form:
   - Title, Description, Category
   - Upload thumbnail via Cloudinary
   - Set price (or free)
   - Add sections and lectures
   - Upload video content
   - Create quizzes
   - Set certificate requirements
3. Submit for review (status: pending)
4. Admin reviews and approves
5. Course goes live (status: published)
```

```javascript
// Progress calculation
const calculateProgress = (course, userId) => {
  const enrollment = course.enrolledStudents.find(e => e.user === userId);
  const totalLectures = course.totalLectures;
  const completedLectures = enrollment.completedLectures.length;
  
  return Math.round((completedLectures / totalLectures) * 100);

// Streak tracking
```

### 10.3 Skill Matching Algorithm

```javascript
// Match score calculation (matching.service.js)
const calculateMatchScore = (userId, matchUser, offeredSkill, desiredSkill) => {
  let totalScore = 0;
  
  // 1. Skill Compatibility (40 points)
  const hasDesiredSkill = matchUser.offeredSkills.includes(desiredSkill);
  const wantsOfferedSkill = matchUser.desiredSkills.includes(offeredSkill);
  totalScore += (hasDesiredSkill && wantsOfferedSkill) ? 40 : 0;
  
  // 2. User Rating (20 points)
  totalScore += (matchUser.rating.average / 5) * 20;
  
  // 3. Experience (15 points)
  totalScore += Math.min(matchUser.completedExchanges * 3, 15);
  
  // 4. Availability (15 points)
  totalScore += matchUser.availability.length > 0 ? 15 : 5;
  
  // 5. Response Rate (10 points)
  totalScore += 10; // Default
  
  return totalScore;
};
```

## 11. Real-time Features

### Socket.IO Events

```javascript
// Socket event handlers (socketHandler.js)
io.on('connection', (socket) => {
  
  // ============ CHAT EVENTS ============
  socket.on('join_room', (roomId) => {
    socket.join(roomId);
  });
  
  socket.on('send_message', (data) => {
    io.to(data.roomId).emit('receive_message', data);
    // Save to database
  });
  
  socket.on('typing', (data) => {
    socket.to(data.roomId).emit('user_typing', data);
  });
  
  // ============ LIVE SESSION EVENTS ============
  socket.on('join_session', (sessionId) => {
    socket.join(`session_${sessionId}`);
    io.to(`session_${sessionId}`).emit('participant_joined', {
      userId: socket.userId
    });
  });
  
  socket.on('leave_session', (sessionId) => {
    socket.leave(`session_${sessionId}`);
    io.to(`session_${sessionId}`).emit('participant_left', {
      userId: socket.userId
    });
  });
  
  socket.on('session_message', (data) => {
    io.to(`session_${data.sessionId}`).emit('new_session_message', data);
  });
  
  // ============ NOTIFICATION EVENTS ============
  socket.on('subscribe_notifications', (userId) => {
    socket.join(`user_${userId}`);
  });
  
  // Server-side notification broadcast
  const sendNotification = (userId, notification) => {
    io.to(`user_${userId}`).emit('new_notification', notification);
  };
});
```

```javascript
// webhook.controller.js
const handleStripeWebhook = async (req, res) => {
  const sig = req.headers['stripe-signature'];
  const event = stripe.webhooks.constructEvent(
    req.body, sig, process.env.STRIPE_WEBHOOK_SECRET
  );
  
  switch (event.type) {
    case 'checkout.session.completed':
      // 1. Update payment status
      // 2. Enroll user in course
      // 3. Update instructor earnings
      // 4. Send confirmation email
      // 5. Create notification
      break;
      
    case 'payment_intent.payment_failed':
      // Handle failure
      break;
      
    case 'customer.subscription.updated':
      // Handle subscription changes
      break;
  }
  
  res.json({ received: true });
};
```

## Summary

SkillVerse is a comprehensive learning platform with:

- **4 User Roles**: Learner, Educator, Skill Exchanger, Admin
- **Core Features**: Courses, Live Sessions, Skill Exchange, Gamification
- **Tech Stack**: MERN + Clerk Auth + Stripe + Socket.IO
- **Real-time**: Chat, Notifications, Live Sessions
- **AI Integration**: Skill Matching, Recommendations

The platform enables:
1. **Learning** through video courses and live sessions
2. **Teaching** by creating and monetizing courses
3. **Exchanging** skills peer-to-peer with matching algorithm
4. **Gamifying** the experience with points, badges, achievements
5. **Certifying** completion with verifiable certificates
